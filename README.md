# Tetris Qwen

A kid-friendly, browser-based Tetris game built as a single `index.html` file.

## Features

- Super Rotation System (SRS) piece rotation
- 7-bag randomizer
- Ghost piece and hold piece support
- Smooth animations
- Classic scoring and level progression
- Local high scores saved with `localStorage`
- Simple oscillator sounds powered by the Web Audio API
- Dark futuristic glassmorphism theme

## Play online

Once GitHub Pages is enabled, play it at:

https://mmhorda.github.io/tetris-qwen/

## Run locally

Clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/mmhorda/tetris-qwen.git
cd tetris-qwen
```

Then double-click `index.html`, or serve it with any static file server.

## Controls

Open the game in a browser and use the on-screen instructions/controls shown by the game. Typical Tetris controls include moving left/right, rotating pieces, soft/hard drop, hold, pause, and restart.
